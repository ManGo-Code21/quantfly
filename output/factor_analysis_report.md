## 二、285因子库 — 11类分组分析

基于聚宽因子库文档，285因子分为以下11类，以下按推荐优先级分组：

### 📊 情绪面因子（Sentiment）

**代表因子：**
- 换手率因子（5日/10日/20日）
- 波动率因子（日收益率标准差、N日收益方差）
- RSI（相对强弱指标）
- 乖离率（BIAS）
- 情绪得分

**初步IC评估：**
| 因子 | IC | 结论 |
|------|-----|------|
| 5日平均换手率 | **-0.040** | 🔻 高换手→跑输，负向 |
| 日收益率标准差 | **-0.046** | 🔻 高波动→跑输，负向最强 |
| 20日收益方差 | **-0.042** | 🔻 波动越大越差 |
| 残差波动因子 | **-0.041** | 🔻 系统性风险补偿 |
| RSI | ≈-0.009 | 极弱，几乎无效 |

**优先测试建议：**
- ❌ 换手率/波动率因子 → 已确认负向，**不推荐**在现有框架单独使用
- ⚠️ RSI → IC 极弱，单独使用无效
- ⚠️ 乖离率 → 需结合趋势方向

---

### 📈 基本面因子（Fundamental）

**代表因子：**
- PE（市盈率）、PB（市净率）、PS（市销率）、PCF（市现率）
- 股息率
- ROE、ROA（盈利能力）
- 净利润增速、营收增速
- 资产负债率

**初步IC评估：**
| 因子 | IC | IR | 结论 |
|------|-----|-----|------|
| **PB（市净率）** | **+0.034** | +0.151 | ✅ 价值有效，低PB→跑赢 |
| **价值因子** | **+0.031** | +0.152 | ✅ 估值偏低→跑赢 |
| PE | 待测 | - | 中证500成分股数据缺失 |
| ROE | 待测 | - | 预计正向（盈利能力） |
| 净利润增速 | 待测 | - | 预计正向（成长性） |
| 营收增速 | 待测 | - | 预计正向（成长性） |
| 股息率 | 待测 | - | 预计正向（稳定现金牛） |

**优先测试建议：**
- ✅ **PB** → 当前最强基本面因子，可立即纳入
- ✅ **PE** → 经典价值因子，需QMT数据验证
- ✅ **ROE** → 盈利能力核心指标
- ✅ **净利润增速** → 成长性核心
- 🔲 **PCF、PS** → 次要估值因子

### 💰 资金面因子（Money Flow）

**代表因子：**
- 北向资金净流入（5日/10日）
- 主力净流入（超大单+大单）
- 小单净流入（反向）
- 主力净流入占比

**初步IC评估：**
| 因子 | IC | 结论 |
|------|-----|------|
| 5日主力净流入均值 | 待测 | 预期正向 |
| 10日主力净流入均值 | 待测 | 预期正向 |
| 超大单净流入 | 待测 | 预期正向 |
| 小单净流入 | 待测 | 🔻 反向（散户买入=卖出信号） |
| 主力净流入占比 | 待测 | 预期稳定正向 |

> ⚠️ QMT API `/data/money_flow` 目前返回空，extended_factor_ic.py 中已定义因子框架，待服务稳定后验证

**优先测试建议：**
- ✅ **主力净流入5日均线** → 资金流向核心
- ✅ **北向资金5日/10日** → 外资情绪指标
- ⚠️ **小单净流入** → 反向指标（散户买→机构卖）

---

### 🚀 动量因子（Momentum）

**代表因子：**
- N日收益率（5日/10日/20日）
- 动量加速度（ret5 - ret10）
- 相对强弱（RS）

**初步IC评估（自研21因子测试结果）：**
| 因子 | IC(50只) | IC(100只) | 结论 |
|------|----------|----------|------|
| ret5 | -0.0004 | -0.009 | 几乎无效 |
| ret10 | -0.009 | +0.001 | 几乎无效 |
| ret20 | -0.019 | **-0.020** | 弱负向（均值回归） |
| ret_accel | +0.012 | -0.013 | 不稳定 |

> 🔍 **关键洞察**：A股短期动量极弱，20日收益呈弱均值回归特征（高久期动量衰减快）

**优先测试建议：**
- ⚠️ **N日收益率** → IC 接近零，不建议单独使用
- 🔲 **动量加速度** → 需配合其他因子
- ❌ 纯动量策略在A股失效，**建议与基本面/资金面结合**

**优先测试建议：**
- ✅ **布林线下轨** → **首选因子，IR=0.315，可立即回测**
