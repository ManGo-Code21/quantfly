## 一、选股策略（5因子评分，动态池）

### 1.1 板块发现（三层信号）

每10个交易日扫描所有板块，计算成分股加权平均动量：

```
综合得分 = 5日动量×0.5 + 10日动量×0.3 + 20日动量×0.2
```

| 等级 | 条件 | 含义 |
|------|------|------|
| **START** | 5日>0% 且 (20日<3% 或 10日<2%) | 🟢 刚启动，底部反转 |
| **ACCEL** | 5日>2% 且 10日>2% | 🟡 加速中，短期确认 |
| **CLIMAX** | 5日>3% 且 20日>5% | 🔴 高潮，不宜追高 |
| WEAK | 5日≤0% | ⚪ 不考虑 |

取综合得分 Top5 板块，进入选股。

### 1.3 个股评分（5因子）

在 Top5 板块的股票池内，对每只股票打分：

```
最终得分 = 动量(40%) + 趋势(30%) + 量价(15%) - 波动惩罚(15%) + 板块加成
```

| 因子 | 权重 | 计算方式 | 分值范围 |
|------|------|---------|---------|
| **动量** | 40% | `20日涨幅×0.7 + 5日涨幅×0.3` | -∞ ~ +∞ |
| **趋势** | 30% | MA5>MA10>MA20 多头=0.20 / 偏多=0.12 / 空头=-0.15 | -0.15 ~ 0.20 |
| **量价** | 15% | 价涨量增+0.15 / 放量下跌-0.10 | -0.10 ~ 0.15 |
| **波动惩罚** | 扣分 | `max(0, (年化波动-50)×0.005)` | 0 ~ -∞ |
| **板块加成** | 加分 | 板块综合得分/100 | |

取 TOP 4 持有。

## 二、择时策略（大盘温度计）

### 2.1 四维市场信号

基于上证指数计算：

| 维度 | 权重 | 指标 | 逻辑 |
|------|------|------|------|
| **趋势** | 50% | MA20 vs MA60 | 价格>MA20>MA60=90分 / 仅>MA20=70 / 仅>MA60=45 / 全破=20 |
| **动量** | 30% | 20日涨幅 | 50 + 涨幅×200（0-100 clamp） |
| **量能** | 20% | 5日vs20日均量比 | 比值×50（0-100 clamp） |

```
temperature = 趋势×0.5 + 动量×0.3 + 量能×0.2
```

### 2.2 辅助信号

| 信号 | 计算 | 用途 |
|------|------|------|
| **量能信号** | `(5日均量/20日均量 - 1) × 2` | 放量=正，缩量=负 |
| **波动率** | 20日对数收益年化波动 | 高波动→减仓 |
| **趋势强度** | `(现价-MA20)/MA20 × 100` | 相对均线偏离 |

## 三、仓位管理（四级递进）

```
temperature < 20  + 放量暴跌 → 0%（黑天鹅紧急空仓）
temperature < 25              → 0%（冰期完全空仓）
temperature < 40              → -25%/次（逐步减仓）
temperature 40-60             → 维持不动
temperature > 60 + 放量+低波  → +20~30%/次（加仓至满仓）
波动率 > 70                   → 仓位减半
```

仓位浮动范围：**0% ~ 100%**，初始 50%。

## 四、风险控制（四道防线）

### 4.1 黑天鹅响应
```
昨日温度>50 且 今日温度骤降至<15
→ 清仓60%，保留40%
```

### 4.2 个股止损止盈

| 类型 | 触发 | 动作 |
|------|------|------|
| **追踪止盈** | 盈利>15%后，从高点回撤8% | 全清（锁利） |
| **硬止损** | 亏损>12% | 全清（截断亏损） |

### 4.3 组合回撤熔断
```
总回撤 > 15%
→ 全部清仓 + 10天冷却期
→ 温度恢复>50 后，40%仓位重新入场
```

### 4.4 波动率过滤
```
市场波动率 > 70 → 仓位减半
（防止高波动环境亏大钱）
```
