## 2. 核心逻辑

### 2.1 牛熊切换

```
if 上证指数 > MA60:
    🐂 牛市 → 全仓 V12.1 策略（0%~100% 动态仓位）
else:
    🐻 熊市 → 仓位上限 30%（防御模式，继续用同策略选股）
```

### 2.2 选股流水线

```
步骤1: 板块动量排序
  └→ 30+ 行业，成分股等权平均 × 5/10/20 日动量
  └→ 分级: CLIMAX / ACCEL / START / WEAK
  └→ 取 TOP 5

步骤2: 逐只评分（6维度）
  └→ 动量 35% + 趋势 25% + 量价 10% + 质量 15% + 情绪 5% - 波动惩罚 + 板块加成
  └→ 取 TOP 4

步骤3: 质量预过滤
  └→ ROE < 5% 或 净利增速 < -20% → 排除
```

### 2.3 仓位温度公式

```
temperature = trend_score × 0.5 + momentum_score × 0.3 + volume_score × 0.2

其中:
  trend_score:  MA5/MA20/MA60 多头排列 → 90，空头 → 20
  momentum_score: 20日收益映射到 [0, 100]
  volume_score:  量比映射到 [0, 100]
```

### 选股权重

| 因子 | 权重 | 计算方式 |
|------|:---:|------|
| 动量 | 35% | 20日收益率 × 0.7 + 5日收益率 × 0.3 |
| 趋势 | 25% | MA多头排列 +0.20 / 空头 -0.15 |
| 量价 | 10% | 放量上涨 +0.15 / 无量下跌 -0.10 |
| 质量 | 15% | ROE + 净利增速 + 营收增速 |
| 情绪 | 5% | 换手率飙升 |
| 波动惩罚 | - | 年化波动 > 50% → 每超1%扣0.005 |
| 板块加成 | + | 板块动量综合分 / 100 |

### 质量评分细则

| ROE | 得分 | 净利增速 | 得分 | 营收增速 | 得分 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| > 15% | +0.10 | > 30% | +0.08 | > 20% | +0.05 |
| 10-15% | +0.05 | 0-30% | +0.03 | 0-20% | +0.02 |
| 5-10% | 0 | < 0% | **-0.15** | < 0% | -0.05 |

5. 点击运行

## 9. 已知局限

1. **牛市策略** — 熊市年收益 -3.2%，不适合单边下跌市场
2. **板块动量依赖** — 需要至少 1-2 个强势板块才能产生有效信号
3. **基本面滞后** — 使用年报数据，季度间可能有信息滞后
4. **K线数据依赖 QMT** — QMT 服务不可用时策略无法运行
5. **北向/融资信号未接入** — 历史数据不可靠，留待数据源修复后启用
